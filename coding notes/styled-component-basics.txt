1. $npm install styled-components

2. 

   import styled from 'styled-components'
   const Button = styled.button`  // <- note the opening backtick 
     background: Bisque;
     font-size: 1em;
     margin: 1em;
     padding: 0.25em 1em;
     border: 2px solid Chocolate;
     border-radius: 3px;
    ` //<-closing backtick

   const Input = styled.input`
     margin: 0.25em;
    `
^This creates styled versions of Button and Input html elements and assigns
them to the variables Button and Input. 

3.
   //in a component's return

 <div>
   <h2>Login </h2>
   <form onSubmit={onSubmit}>
     <div>
       Username:
       <Input /> //the variable styled in step 2 above
     </div>
     <div>
       Password:
       <Input  type='password'/>
    </div>
    <Button type='submit' primary=''> Login </Button>
  </form>
 </div>
