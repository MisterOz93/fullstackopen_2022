if using Heroku:

1. $npm install -g heroku //Installs it globally w/ node.js, not just in
			   // a specific dir

2. create file called Procfile to backend root
containing the line:
web: node index.js 

3. change PORT definition to: 
  const PORT = process.env.PORT || 3001 //No need to write anything in .env file

4. $heroku create
 
5. $git push heroku main <-- after pushing updates to github
