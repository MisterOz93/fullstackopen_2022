1. app root has db.json file
2. in root dir of app run: $npx json-server --port 3001 --watch db.json

^this launches the server, instead do:

3. Install json-server as a development dependency 
        (only used during development) by executing the command:
	$npm install json-server --save-dev (in the root dir of app)
4. make following addition to package.json file inside scripts object:
	"server": "json-server -p3001 --watch db.json"
5. Can now start the json server from app's root dir w/ command:
	$npm run server
